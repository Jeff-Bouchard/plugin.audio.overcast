Overcast for Kodi
=====================

[![CircleCI](https://circleci.com/gh/brianhornsby/plugin.audio.overcast/tree/develop.svg?style=svg)](https://circleci.com/gh/brianhornsby/plugin.audio.overcast/tree/develop)
